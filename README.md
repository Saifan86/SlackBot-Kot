# "/" commands

* **👑 /kot-roast `@user`** – KOT uses the Gemini API to royally insult/roast a user in a funny, medieval style (without being genuinely mean).
* **📜 /kot-decree** – KOT issues a dramatic, random royal decree regarding tech, coding, or life as a bot.
* **💡 /kot-idea** – Generates a unique, creative, and beginner-friendly coding project idea for a teen developer.
* **⚔️ /kot-duel `@user`** – Challenges another user to a dramatic, medieval-style coding battle with a randomly generated prompt.
* **⚡ /kot-motivate** – Hypes up a stuck or tired programmer with an energetic, royal motivational speech.
* **👋 /kot-hello** – Greets users properly (as the peasants they are).
* **🏓 /kot-ping** – Quick health check to ensure the King is awake.
* **📢 /kot-standup `[text]`** – Formats and logs your daily progress directly to the channel.
* **📊 /kot-poll `[question]`** – Quickly launches a survey with thumbs up, down, and neutral reaction guides.
* **👑 /kot-crown `@user`** – Publicly crowns someone as the "Coder of the Day." All shall bow!

# 👑 KOT (King Of Bots)

KOT is an AI-powered, royalty-themed Slack bot built for the **Hack Club Stardance** summer program. Using the free Google Gemini API, KOT rules over your Slack workspace, offering medieval-style roasts, royal decrees, project generation, and interactive coding challenges.
